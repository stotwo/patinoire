# La Patinoire ⛸️

Bienvenue sur le site officiel de **La Patinoire**, le complexe sportif sur glace de référence. Ce site vitrine présente toutes les informations essentielles pour nos visiteurs et nos futurs patineurs, dans un design moderne, épuré et immersif.

## À propos du site

Le site web a été conçu pour offrir une expérience utilisateur fluide et agréable, mettant en avant nos différentes activités sur la glace :

- **Séances publiques** : Horaires et tarifs pour venir patiner librement.
- **Cours de patinage** : Informations pour les enfants et les adultes souhaitant apprendre ou se perfectionner.
- **Affûtage des patins** : Présentation de notre service technique (Standard et Express).
- **Anniversaires** : Formules de groupe (Givré, Frisson, Glaçon) pour fêter un événement sur la piste.
- **Galerie & Historique** : Découverte des lieux en images et rétrospective depuis l'inauguration en 1973.
